## 内置的 web 服务器 {#builtin_web_server_title}

PHP 5.4 之后, 你可以不用安装和配置功能齐全的 Web 服务器，就可以开始学习 PHP。
要启动内置的 Web 服务器，需要从你的命令行终端进入项目的 Web 根目录，执行下面的命令:

{% highlight console %}
> php -S localhost:8000
{% endhighlight %}
