## 내장 웹 서버 {#builtin_web_server_title}

PHP 5.4 이상의 버전은 다른 웹서버의 설치와 설정 없이 PHP를 바로 시작할 수 있습니다. 웹서버를 실행하려면 여러분의
프로젝트 폴더에서 다음과 같은 커맨드를 실행하면 됩니다.

{% highlight console %}
> php -S localhost:8000
{% endhighlight %}
